# Campus Placement System

## Project Overview
The Campus Placement System is designed to streamline the process of campus placements for students and companies alike. The system provides an efficient platform where students can apply for jobs and companies can post job listings.

## Technologies Used
- HTML5  
- CSS3  
- Python  
- MySQL  
- JavaScript

## Features
- User Registration and Login  
- Job Listings by Companies  
- Application Submission by Students  
- Admin Panel for Managing Users and Job Postings  
- Search Functionality for Jobs  
- Notifications for Application Status

## Installation Instructions
1. Clone the repository:  
   ```bash
   git clone https://github.com/Rishabh2619/campus-placement-system.git
   ```  
2. Navigate to the project directory:  
   ```bash
   cd campus-placement-system
   ```  
3. Install the required packages:  
   ```bash
   pip install -r requirements.txt
   ```  
4. Set up the MySQL database and configure the database credentials in the config file.
5. Run the application:  
   ```bash
   python app.py
   ```

## Usage Guidelines
- Register as a student or company to access the features.  
- Follow the on-screen instructions to post jobs or apply for available positions.  
- Check the application status through the dashboard.
